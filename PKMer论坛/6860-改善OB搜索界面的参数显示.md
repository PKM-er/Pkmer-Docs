# 改善OB搜索界面的参数显示

![251021_改善OB搜索界面的参数显示-img-251021_114834.webp](https://cdn.pkmer.cn/original/1X/8d1567288665bcf72f5010317ca0a2c7eb6f8a9a.webp)
左：更新前，每行一个选项
右：更新后，选项集中在一行

这样一来，即使常态展开搜索选项也不会占用过多空间

CSS 代码：
```css
/* 搜索选项 */
.search-params {
    display: flex;
    /* 平均分 */
    justify-content: space-around;

    /* 隐藏原始文本 */
    .setting-item-info .setting-item-name {
        display: none;
    }

    /* 自定义显示文本 */
    .setting-item .setting-item-info {
        margin-right: 0.5em;
        &::before {
            font-size: var(--font-ui-small);
        }
    }

    .setting-item:nth-child(1) .setting-item-info::before {
        content: "折叠结果";
    }

    .setting-item:nth-child(2) .setting-item-info::before {
        content: "显示上下文";
    }

    .setting-item:nth-child(3) .setting-item-info::before {
        content: "搜索说明";
    }
}
```
