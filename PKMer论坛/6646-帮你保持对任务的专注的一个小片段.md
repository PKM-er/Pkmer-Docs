## 说明
这是一个非常简单的 CSS Snippet，作用是：**让处于注释中的任务清单变得模糊。**

它看起来就像是这样：
![250912_帮你保持对任务的专注的一个小片段-img-250912_192416.webp](https://cdn.pkmer.cn/original/1X/0c82b72daa49d86d408ad088db30714d43ccf915.webp)

你需要做的就是，把一部分（暂时不需要在意的）任务放进两个 `%%` 行中间，让它们成为注释。

然后应用这个 CSS 片段：

```css
/* 注释中的任务 */
.HyperMD-list-line:not(.cm-active) .cm-comment,
.HyperMD-task-line:not(.cm-active) .cm-comment {
    opacity: 0.3;
    filter: blur(2px);

    &:hover{
        opacity: 1.0;
        filter: none;
    }
}
```

在你聚焦的时候，他们会显示出原始文本。
