# Zotero 笔记样式——袖珍 Blue Topaz

## Zotero 笔记界面 CSS 使用方法

1. 复制 css 文件中的代码
2. 打开 Zotero 设置，转到「高级」，滑到最下面点击「编辑器」按钮
3. 接受风险
4. 在搜索栏中输入 `note.css`
5. 将复制的 css 代码粘贴到右侧栏中，完成设置

![Zotero 笔记样式——袖珍 Blue Topaz](https://cdn.pkmer.cn/images/202311302343994.png!pkmer)
