# 自定义 Excalidraw 脚本 -Frame 悬浮大纲（FrameFloatingOutline）

### 脚本介绍

**FrameFloatingOutline** 是一个专为 Excalidraw 设计的 Frame 悬浮大纲工具，提供轻量级的 Frame 管理和排序功能。用于简单的 Frame 列表展示和手动排序，线性布局。

| 界面组件      | 功能描述         | 主要特性                                                  |
| --------- | ------------ | ----------------------------------------------------- |
| **标题栏**   | 面板顶部控制区域     | • 显示 "Frame 大纲 " 标题<br>• 包含 Frame 数量统计<br>• 拖拽可移动面板位置 |
| **操作按钮栏** | 全局控制按钮       | • `🔢` 确认保存排序按钮<br>• `🔄` 手动刷新大纲按钮<br>• `✖` 关闭悬浮大纲按钮  |
| **已排序区**  | 显示已编号的 Frame | • 按 01、02、03 顺序排列<br>• 支持拖拽重新排序<br>• 显示序号标签           |
| **未排序区**  | 显示未编号的 Frame | • 显示为 "--" 序号<br>• 可拖入已排序区进行编号<br>• 与已排序区用虚线分隔        |
