## 概述

### Lskypro Upload V2 插件总结
1. **主要功能**：自动将本地图片上传至 LskyPro 图床，支持自动上传并替换粘贴的图片为 URL，具备多版本 API 支持、多种上传方式、批量上传、并发上传、智能处理图片及多语言支持等功能。
2. **适用场景**：适用于使用 Obsidian 且有图片上传至 LskyPro 图床需求的用户，无论是桌面端还是移动端都可使用。
3. **核心特色**：轻量高效，兼容 LskyPro V1 / V2 API，有多种上传途径，支持批量和并发上传，还能对网络图片进行处理及自动删除源文件，内置中英双语切换。
4. **使用建议**：使用前需确保有运行且可访问的 LskyPro 图床服务，开启 API 访问，获取有效访问令牌，V2 版本还需有效存储 ID。在 Obsidian 设置中完成插件基本配置，可按需进行高级选项设置。
