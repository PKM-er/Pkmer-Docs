# Calctex

一个 [Obsidian](https://obsidian.md/) 插件，可以自动计算您的 LaTeX 公式的解决方案。

使用方法

要计算一个公式的解，只需在公式后面加上一个“=”符号，解将自动计算并显示出来。如果你想将解写入文档中，只需点击解即可。

## 安装

1. 从发布页面下载最新版本。
2. 将 zip 文件解压到 `<vault>/.obsidian/plugins/` 目录下。
3. 重新加载 Obsidian 并在设置中启用插件。
4. 尽情享受吧！
