# Obsidian Autoscroll 插件

## 特点

- 添加了一个按钮来滚动打开文件的内容
- 右键单击按钮以更改滚动速度

# 已知问题

- 如果在点击文件内容之前按下按钮，则无法工作（插件会滚动浏览已打开且聚焦的文件）
- 插件设置中的速度设置正确，但当前速度在设置菜单中显示不正确
