# 选择当前行插件

![Apr-13-2023 15-22-57](https://user-images.githubusercontent.com/8376313/231723638-46cec13b-3b7e-4962-9ecb-f5b1487c67cc.gif)

一个简单的插件。

当你按下键盘上的 `ESC` 按钮时，当前行会被选中。从行的开头到结尾。

注意：安装后可以将热键更改为其他特定的键组合。
