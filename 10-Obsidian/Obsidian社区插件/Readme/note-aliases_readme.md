# 注意别名

该插件允许在Obsidian中对笔记别名进行精细管理。

计划了几个功能。目前只有一个功能可用，但它是必不可少的。

## 命令

|ID|名称|描述|
|--|----|-----------|
|save-alias|在目标笔记的前置元数据中保存光标下链接的别名|当光标位于带有别名的wikilink上时，此函数会查找或创建此链接的目标笔记，并将此别名添加到该笔记的YAML前置元数据中的别名列表中|
