# Obsidian的Wordle插件

**注意：该插件已不再积极维护。**

这是一个Obsidian插件，允许你直接从编辑器中玩Wordle游戏！
它已经针对Obsidian **v0.13.19**进行开发，并在**Windows**上进行了测试。

## 安装

从**发布**页面获取最新版本，并将所有文件复制到`.obsidian/plugins/obsidian-wordle/`。就是这样！
当获得批准后，您还可以通过在Obsidian中搜索**wordle**来安装它。
