# Obsidian Wikidata 导入器

该插件将数据从 Wikidata 数据库导入到您的 Obsidian 笔记中。

您只需要在笔记中添加一个 `wikidata实体id` 属性，

然后运行 `Wikidata导入器：导入活动文件的属性` 命令，

就可以了。您的笔记现在具有所有 Wikidata 属性作为笔记的属性。
