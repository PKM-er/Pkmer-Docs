# 语音转文本键盘助手

**注意：**此插件不提供语音转文本服务

由于 Google 语音转文本 Android 键盘无法处理“新行”命令，此插件通过 Obsidian 命令面板提供了这样的命令（并且可以将其添加到移动按钮工具栏中）。

目前支持的命令：

- 追加新行（并跳转到该行）
- 插入新行（并跳转到该行）
