# Obsidian转录

一个通过[OpenAI Whisper](https://openai.com/blog/whisper/)生成任何音频和视频文件高质量转录的插件。

![演示](media/demo.gif)

## 特点

- 通过[ffmpeg](https://ffmpeg.org/)支持广泛的音频和视频文件格式。
- 灵活的转录引擎 - 云端或本地
  - [Whisper ASR](https://github.com/ahmetoner/whisper-asr-webservice)
- 每行转录的开始和结束时间戳

## 如何使用

1. 安装插件
2. 注册一个账户。
3. 从仪表板复制 API 密钥，并将其粘贴到插件的设置中
4. 转录任何音频或视频文件！
