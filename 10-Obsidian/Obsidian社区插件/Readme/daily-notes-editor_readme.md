# 每日笔记编辑器

这是一个插件，可以让您在一个页面上（内联方式）编辑一堆每日笔记，其工作方式类似于 Roam Research 的默认每日笔记视图。

![每日笔记视图](https://cdn.pkmer.cn/covers/daily-notes-editor_1_0.gif)

# 特点

- 在一个页面/内联中打开每日笔记编辑器页面，以便编辑一堆每日笔记。
  - 您可以通过命令打开它。
  - 您可以从左侧功能区打开它。
