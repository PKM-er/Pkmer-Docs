## 命令

`Compare`: 比较您的活动文件与另一个文件，并查看差异。

`Compare and merge`: 与 `Compare` 相同，但您可以选择合并更改。

`Find sync conflicts and merge`: 查找所有 Syncthing 冲突，并按顺序显示每个冲突的合并选项。
