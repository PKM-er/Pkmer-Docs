# Discordian 插件

Discordian 插件是 [Discordian主题](https://github.com/radekkozak/discordian) 的附属插件，允许您从 Obsidian 设置面板自定义它。

使用 Discordian 主题不需要此插件，但**强烈推荐**使用。

## 可用设置

- 编写模式
- 平坦安迪模式
- 可自定义淡出的段落焦点
- 可读的行长度
- 暗色笔记标题（作为默认选项）
- 可自定义的笔记和文件浏览器字体大小
