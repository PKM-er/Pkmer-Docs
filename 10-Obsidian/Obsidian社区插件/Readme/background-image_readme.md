# Obsidian编辑器背景插件

该插件旨在为Obsidian的编辑器视图提供美观的背景。目前它只支持远程资源，所以您需要自己托管图像。

作为设置的一部分，还可以进行其他调整，比如修改背景的模糊度和输入区域的对比度。

我使用动态gif作为我的背景，特别是[waneella](https://waneella.com/)的作品。
