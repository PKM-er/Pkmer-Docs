## 安装

按照常规的 [社区插件](https://help.obsidian.md/Advanced+topics/Community+plugins) 安装。

### 从 Obsidian 内部

从 Obsidian v0.9.8 开始，您可以通过以下步骤在 Obsidian 内部激活此插件：

- 打开设置 > 第三方插件
- 确保安全模式处于**关闭**状态
- 点击浏览社区插件
- 搜索此插件
- 点击安装
- 安装完成后，关闭社区插件窗口并激活新安装的插件
