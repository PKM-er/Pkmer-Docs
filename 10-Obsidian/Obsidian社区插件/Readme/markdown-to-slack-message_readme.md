# Markdown转换为Slack消息

该插件将Markdown笔记转换为Slack消息块，使您能够发送到您的Slack。
# 指南

1. 打开一个笔记
2. 使用快捷键 Command + P 并搜索命令“Markdown to Slack Message”
3. 运行该命令

它将打开 Slack Block Kit 构建器网页，并显示您的内容。因此，您可以使用您的 Slack 账户发送消息。

v0.0.1
