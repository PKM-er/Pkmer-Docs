# Obsidian 简单画布搜索插件

只需安装插件，为搜索模态操作设置一个快捷键，然后您就可以使用它在画布页面中进行模糊搜索和导航。

# 设置

您可以仅搜索笔记标题，或在模糊搜索中包含笔记内容，但如果您的画布非常大，这可能会减慢搜索速度，因为它并没有进行任何优化。
