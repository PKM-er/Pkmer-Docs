## Obsidian前置元数据标签建议

在YAML前置元数据中自动补全标签。如果你在以`tags:`或`tag:`开头的行上，它将显示一个自动补全菜单。

这个插件很有用，因为Obsidian只在使用#字符时显示标签自动补全，但如果你在`tags:`前置元数据中使用#字符，文件将不会被视为属于该标签。这个插件消除了必须返回并删除任何默认自动补全标签上的#字符的需要。
