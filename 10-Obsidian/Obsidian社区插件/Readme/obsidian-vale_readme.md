## 故障排除

Obsidian Vale目前不支持Obsidian 13中引入的新编辑器。

要继续在版本13中使用该插件，您可以启用传统编辑器。

1. 打开**设置** -> **编辑器**。
2. 启用**传统编辑器**。
3. 重新启动Obsidian。
