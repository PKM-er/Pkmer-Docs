# TickTick for Obsidian

TickTick是一款功能强大的待办事项和任务管理应用，可以在所有设备上实现无缝的云同步。

该插件与[TickTick](https://ticktick.com)集成，使您可以轻松地在TickTick和Obsidian之间转换和同步数据。
使用方法

要使用该插件，您首先需要通过插件设置登录到TickTick。完成后，您可以根据需要访问命令。
## 特点

- 创建任务到TickTick
## 开发者信息
