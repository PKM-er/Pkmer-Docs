## 包

该存储库包含多个包，以确保关注点得到清晰分离。

- `packages/data-entry` 包含用户在其设备上安装的插件。
- `packages/obsidian-mocks` 包含用于复制 Obsidian 功能的模拟对象，因为 Obsidian 团队提供的包仅包含用于测试的类型，而没有任何实现。
