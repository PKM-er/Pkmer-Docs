# 复制行（s）

命令用于向下或向上复制行（s），以及向右或向左复制选择内容

您可以使用多光标，选择内容将被保留

将您自己的快捷键设置为 <kbd>Shift Ctrl ↓</kbd> <kbd>Shift Ctrl ↑</kbd> <kbd>Shift Ctrl ←</kbd> <kbd>Shift Ctrl →</kbd>

默认情况下，在 Obsidian 中，如果没有任何选择内容，您可以按下 <kbd>Ctrl C</kbd>，然后按下 <kbd>Ctrl V</kbd> 来复制光标所在的行。
