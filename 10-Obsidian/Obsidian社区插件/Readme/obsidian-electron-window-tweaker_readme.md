## Obsidian Electron 窗口调整器

调整各种 Electron 窗口设置。

目前支持的功能：
- 窗口不透明度
- 窗口模糊效果（在 MacOS 上）
- 窗口始终置顶

*注意：并非所有设置在所有操作系统上都有效*
